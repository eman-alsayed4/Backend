## at my server you can:
* login to the system
* register to the system
* logout from the system
* create a user / card
* edit a user / card
* delete a user / card
* search a user / card
* get all users / cards
* get a user / card by id
* get a tasks 
* Add a task to a specific grade
* Upload a file
